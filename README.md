# Spam Detection using Machine Learning

## Project Description
This project detects whether a message is Spam or Ham using Machine Learning.

## Algorithms Used
- Logistic Regression
- Naive Bayes

## Project Files
- imports.py
- preprocessing.py
- logistic_regression_model.py
- naive_bayes_model.py
- main.py

## Requirements

Install the required libraries:

```bash
pip install pandas numpy scikit-learn
```

## Dataset

Place the `spam.csv` dataset in the project folder before running the project.

## Run the Project

```bash
python main.py
```

## Output

The project displays:
- Accuracy
- Precision
- Recall
- F1 Score
- ROC-AUC Score
- Confusion Matrix
- Classification Report
